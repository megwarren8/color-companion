# Color Companion: pick the right marker for the color you see

An offline single-page tool that turns a wall of Ohuhu markers into a searchable, sortable, tap-friendly reference. Point it at a color you like, and Color Companion tells you which marker in the set matches it most closely, side by side. Built for painters and illustrators who work from photo references and want to stop rummaging.

Designed and built by [Megan Warren](https://github.com/megwarren8).

## What it does

- **Match a color.** Enter a hex, RGB, or hue and Color Companion ranks every marker by how close it comes, closest first.
- **Browse the whole set.** Every marker in the collection, with its swatch, name, and code. Sort by hue, lightness, or family.
- **Filter as you go.** Narrow by color family, warmth, or saturation without breaking the layout.
- **Print a color match card.** One click opens a print window with the marker's swatch and code ready for the shelf.

## Run it

Open `color-companion.html` in any modern browser. Everything (the marker database, the color math, the interface) is baked into the one file, so it works offline the moment it opens.
